GitHub Gist sync — access from any device

How to set up cross-device sync

Your data syncs to a private GitHub Gist — no third-party servers, free forever.

  1. Go to github.com → Settings → Developer settings → Personal access tokens → Tokens (classic)
  2. Click "Generate new token (classic)" and give it a name like "PropTrack"
  3. Check only the gist scope
  4. Copy the token (starts with ghp_) and paste below
  5. Save settings, then click Sync — a Gist ID will be created automatically
  6. On any other device, enter the same token + Gist ID to load your data
